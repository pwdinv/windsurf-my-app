# OpenRouter API Key Configuration
#
# 1. Create a file named .env.local in the project root
# 2. Add the following lines with your actual API keys:
#
# OPENROUTER_API_KEY=your_openrouter_api_key_here
# OPENROUTER_API_KEY_DRAFTER=your_drafter_api_key_here
#
# 3. Get your API keys from: https://openrouter.ai/keys
#
# IMPORTANT: Never commit .env.local to git - it is already gitignored.
